Приложение для учёта тренировок и прогулок. Добавляйте, редактируйте и удаляйте записи о пройденных километрах.

## Возможности
- Добавление тренировок с датой и дистанцией
- Автоматическая сортировка по дате
- Суммирование расстояния при совпадении дат
- Редактирование записей
- Удаление записей

## Стек
- React
- TypeScript
- Vite

## Установка и запуск
```bash
yarn install
yarn dev
